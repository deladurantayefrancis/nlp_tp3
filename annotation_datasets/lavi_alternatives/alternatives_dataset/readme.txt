This archive contains the data set of subject plausibility ratings of alternatives to conversationally negated predicates described in:

G. Kruszewski, D. Paperno, R. Bernardi and M. Baroni. There is no logical negation here, but there are alternatives: Modeling conversational negation with distributional semantics. To appear in Computational Linguistics.

The distributed files contain average subject ratings for negated-item/alternative pairs in the IT and THERE sentential contexts.

Please refer to the paper for details about how the data were collected, and cite it in published work that uses the data set.
